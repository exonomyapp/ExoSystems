# Exonomy Desktop (Flutter Client)

This is the heavy UI desktop Flutter client for Exonomy, designed for managing deep project funding trees and voucher ledgers.

## Architecture & Topology
For a complete understanding of how this desktop client fits into the ecosystem, please refer to:
[Exonomy Topology Specification](../docs/spec/23_exonomy_topology.md)
