# Conscia Web (Index & Federation Dashboard)

This is the Flutter Web application for browser-based node administration, served directly by the Conscia daemon.

## Key Features
- Remote node administration
- Public metadata indexing search
- Federation status and metrics dashboard

## Architecture & Topology
For a complete understanding of how this indexing node fits into the triad, please refer to:
[Spec 22: Application Triad Architecture](../docs/spec/22_application_triad_architecture.md)
