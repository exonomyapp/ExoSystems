# Exocracy Desktop (Flutter Client)

This is the heavy UI desktop Flutter client for Exocracy project managers requiring complex layouts (Gantt charts, deep organizational views, voucher milestone mapping).

## Architecture & Topology
For a complete understanding of how this desktop client fits into the ecosystem, please refer to:
[Exocracy Topology Specification](../docs/spec/24_exocracy_topology.md)

## Development

```bash
flutter run -d linux
```
