# Conscia Management Console (CMC)

The Conscia Management Console (CMC) is a high-density, pure-Flutter desktop and web application designed for comprehensive node governance.

## Architecture
- **Frontend**: Pure Flutter
- **Identity Gateway**: Consumes the shared `exoauth` package to enforce the "Solid Front Door" identity verification standard before granting access to node telemetry.
- **Purpose**: Provides complex, high-resolution interfaces for managing node capabilities, peer delegation, and monitoring real-time mesh traffic.
