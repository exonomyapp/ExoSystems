# Conscia Lite (Mobile Monitoring Client)

This is the mobile-first Flutter client for on-the-go node monitoring and quick governance actions.

## Key Features
- Node health status (Online/Offline/Sleeping)
- Push notification for incoming petitions
- Quick-approve/deny governance actions
