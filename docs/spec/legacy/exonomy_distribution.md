# Exonomy Distribution & User Experience Specification

This document defines the distribution strategy for Exonomy, the sovereign voucher-minting wallet and social media application.

## 1. Philosophical Identity
Exonomy is the "Social Ledger" of the mesh. It must be as portable as physical cash while maintaining the cryptographic rigor of a sovereign node.

## 2. Platform Distribution Matrix

| Platform | Format | Channel | Purpose |
| :--- | :--- | :--- | :--- |
| **Android** | **APK / AAB** | Play Store / F-Droid | Primary platform for mobile commerce and social interaction. |
| **iOS** | **IPA** | App Store | Essential for mass-market social reach. |
| **Linux/Windows** | **Flutter Desktop** | Flatpak / MSIX | Used for administrative voucher management and merchant dashboards. |

## 3. Deployment Methods
- **Consumer**: App Store / F-Droid installation.
- **Merchant**: Docker-based headless wallet for automated voucher processing.
- **Enterprise**: Sideloaded APKs for controlled organizational voucher systems.
