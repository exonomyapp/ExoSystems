# Commercial Reporting (Conscia)

## 2.4 Commercial Reporting (Conscia)

ExoTalk distinguishes between basic communication and advanced data intelligence:

- **Archive-as-Lifeline (Free):** Basic message persistence and P2P synchronization are provided as a core network utility at no cost.
- **Reporting Services (Commercial):** Advanced data aggregation, metrics (views, upvotes), and longitudinal reporting are provided by **Conscia** nodes as a paywalled service.
- **Conscire Agreement:** Access to the reporting layer is gated via a cryptographic service agreement, triggered by the **Conscire** action in the UI.
- **Publisher‑Led Aggregation:** Raw receipts are collected by Conscia nodes which then publish "Squashed" verified totals for a fee.
