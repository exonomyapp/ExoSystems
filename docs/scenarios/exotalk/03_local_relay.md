# Scenario 03: The Local Relay

## Narrative Context
**Malik**, working from a solar-powered bunker in Beirut, realizes that Isabella's local connection is fragile. He initializes a Conscia node to act as a "Sovereign Lifeline." He provides a stable relay point for the community of defenders.

## Cognitive Shift
**Centralized Expectation**: "I’ll use the cloud to store my messages."
**Sovereign Reality**: "We use local, community-owned relays to ensure uptime when the state cuts the fiber."

## World Event Reference
The weaponization of the national grid and the rise of "Solar-Powered Sovereignty" in Lebanon and Gaza.

## Technical Milestones
- Initializing a Conscia node on a separate machine.
- Binding the ExoTalk client to the local relay.
- Verifying the "Direct Link" status in the UI.
