# Scenario 06: Offline Propagation

## Narrative Context
The state security forces cut the fiber optics in Isabella’s district. Isabella has a critical war crimes report to send. She uses her ExoTalk client to "drop" the message into a local node. The report is carried out of the blackout zone on a **Doctors Without Borders** volunteer’s tablet, syncing with the global mesh the moment it hits a mobile signal.

## Cognitive Shift
**Centralized Expectation**: "I need the internet to send a message."
**Decentralized Reality**: "The mesh is physical. Store-and-forward via 'sneakernet' and local relays ensures the truth gets out."

## World Event Reference
The repeated total internet blackouts in Gaza and the reliance on physical data transfer for survival.

## Technical Milestones
- Creating an "Offline" message.
- Synchronizing via a local "Courier" node (sneakernet simulation).
- Verifying propagation to the global mesh once connectivity is restored.
