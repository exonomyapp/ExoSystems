# RepubLet Indexing Node (Web)

This is the Rust backend and Flutter Web dashboard for the RepubLet Blind Indexing Service. It acts as an academic citation relay and public search gateway for open-science metadata.

## Architecture & Topology
For a complete understanding of how this indexing node fits into the ecosystem, please refer to:
[RepubLet Topology Specification](../docs/spec/25_republet_topology.md)

## Development
```bash
# Rust backend
cargo run

# Flutter Web dashboard
flutter run -d chrome
```
