# Exocracy Indexing Node (Web)

This is the Rust backend and Flutter Web dashboard for the Exocracy Blind Indexing Service. It provides high-throughput search indexing for open tasks and bounties across the mesh.

## Architecture & Topology
For a complete understanding of how this indexing node fits into the ecosystem, please refer to:
[Exocracy Topology Specification](../docs/spec/24_exocracy_topology.md)

## Development
```bash
# Rust backend
cargo run

# Flutter Web dashboard
flutter run -d chrome
```
